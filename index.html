css is a language we use to style a web page.

3 reasons why we use css

  1. css is used to define styles for your webpages, including the design, layout and variations in display for different devices and screen sizes.
  2. css describes how HTML elements are to be displayed on screen, paper, or in other media.
  3. css saves a lot of work. It can control the layout of multiple webpages all at once.
